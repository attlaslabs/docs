Lorsque vous créez un chat sur Attlas, vous rédigez votre contenu dans votre propre langue : le nom de votre chat, sa description, le message de bienvenue et les libellés de vos déclencheurs.

Attlas traduit automatiquement tout ce contenu afin que chaque visiteur voie votre chat dans sa propre langue, sans aucun effort supplémentaire de votre part.

---

## Fonctionnement

Lorsque vous enregistrez un déclencheur ou mettez à jour le profil de votre chat, Attlas traduit votre contenu en 13 langues en arrière-plan.

Lorsqu’un visiteur ouvre votre chat, il voit tout, y compris les libellés de déclencheurs et votre message de bienvenue, dans sa propre langue.

Si un visiteur clique sur un déclencheur de question, la question est envoyée à l’IA dans sa langue, de sorte que la réponse revienne également dans sa langue.

## Détection de la langue

Attlas détecte la bonne langue pour chaque visiteur dans cet ordre :

1. **Choix manuel** — si le visiteur a sélectionné une langue à l’aide du sélecteur de langue, ce choix est toujours respecté.
2. **Langue de la conversation** — la langue dans laquelle le visiteur écrit réellement.
3. **Langue du navigateur** — la langue configurée dans son navigateur.
4. **Localisation** — si le navigateur ne donne aucun signal clair, Attlas utilise le pays du visiteur comme solution de repli.
5. **Anglais** — la valeur par défaut si rien d’autre n’est détecté.

## Sélecteur de langue

![Sélecteur de langue](/images/triggers-translation-list.webp)

Chaque chat dispose d’un sélecteur de langue dans l’en-tête (icône du globe). Les visiteurs peuvent l’utiliser pour basculer à tout moment vers l’une des 13 langues prises en charge.

Leur choix est enregistré pour les visites futures.

## Ajustement des traductions

Si une traduction automatique vous semble incorrecte pour une langue spécifique, vous pouvez la modifier directement, sans toucher à votre contenu source.

<Info>
  Cette option n’est visible que par vous, en tant que propriétaire du chat, dans votre espace de travail. Les visiteurs ne voient jamais le bouton d’édition.
</Info>

**Pour modifier une traduction :**

![Accéder aux modifications manuelles](/images/triggers-translation-button.webp)

1. Ouvrez votre chat dans l’aperçu de la barre latérale de l’espace de travail.
2. Cliquez sur l’**icône de drapeau** dans l’en-tête du chat pour ouvrir le sélecteur de langue.
3. Cliquez sur **Modifier les traductions** (icône de crayon) en haut du menu déroulant.

![Accéder aux modifications manuelles](/images/triggers-translation-editor.webp)

4. Sélectionnez la langue que vous souhaitez ajuster.
5. Modifiez les champs de votre choix : nom du chat, description, message de bienvenue, ou libellés et questions des déclencheurs.
6. Cliquez sur **Enregistrer**.

Les modifications sont enregistrées immédiatement et prennent effet la prochaine fois qu’un visiteur ouvre le chat dans cette langue.

<Tip>
  Utilisez la barre de recherche dans l’éditeur pour trouver rapidement un déclencheur spécifique lorsque vous en avez beaucoup.
</Tip>

## Aucune configuration requise

Le support multilingue est actif pour tous les chats sur tous les forfaits. Il n’y a rien à configurer ni à activer.

Votre contenu est traduit automatiquement chaque fois que vous enregistrez un déclencheur ou mettez à jour le profil de votre chat.

<Tip>
  Rédigez votre contenu naturellement dans votre propre langue. Attlas s’occupe des traductions.
</Tip>
