Une fois votre chat configuré et entraîné, il est temps de le partager avec le monde.

Attlas propose plusieurs méthodes de distribution afin que vous puissiez choisir celle qui correspond le mieux à votre audience, canal ou projet.

Pour accéder à ces options, cliquez sur le bouton **<kbd>Share</kbd>** situé dans le coin supérieur droit de la plateforme.

![Sharing Options 1](/images/sharing-options.webp)

---

## Options de partage

Voici les 4 principales façons de partager votre chat :

1. **Lien direct** : Parfait pour les réseaux sociaux, les e-mails et les applications de messagerie.
2. **Scanner et discuter (QR Code)** : Idéal pour les supports imprimés et les emplacements physiques.
3. **Bulle de chat** : Un bouton flottant en bas de votre site web.
4. **Afficher sur une page (Iframe)** : Un widget intégré directement dans vos pages.

---

## Lien direct

Le lien direct vous permet de partager une version pleine page de votre chat. Vous pouvez l'utiliser sur diverses plateformes :

- **Réseaux sociaux** : Partagez des liens directement dans les publications ou les liens bio (bio Instagram, LinkedIn, Linktree).
- **Messagerie** : Envoyez directement via WhatsApp, SMS ou Telegram.
- **E-mail** : Ajoutez aux newsletters, campagnes ou signatures d'e-mails.
- **Supports promotionnels** : Ajoutez aux présentations, PDF ou documents.

Copiez simplement le lien public généré depuis votre tableau de bord et partagez-le avec votre audience.

---

## Scanner et discuter (QR Code)

Connectez le monde physique à votre chat IA en générant un QR Code personnalisé. Il est très efficace pour :

- **Supports imprimés** : Cartes de visite, flyers, catalogues et livrets.
- **Emplacements physiques** : Menus de restaurant, tables, emballages ou affiches.
- **Événements** : Bannières, stands de salon et présentations.

Une fois scanné, les utilisateurs seront redirigés directement vers votre chat dans leur navigateur mobile.

<Info>
  Si vous modifiez votre nom d'utilisateur dans la section **Profil**, vous devrez générer et télécharger un nouveau QR Code.
</Info>

## Ajouter une bulle de chat

Ajoutez une bulle de chat flottante dans le coin inférieur de votre site web afin que les visiteurs puissent interagir avec votre assistant sans quitter votre site.


 ![Attlas Bubble Chat Site](/images/attlas-bubble-chat-site.jpg)


<Note>
  L'option de partage par bulle de chat est disponible exclusivement sur les **plans Attlas Starter et PRO**.
</Note>

### Installation

Copiez le snippet de script et collez-le juste avant la balise fermante `</body>` de votre HTML :

```html
<script 
  id="attlas-widget" 
  data-chat-id="YOUR_CHAT_ID" 
  src="https://attlas.so/embed.js" 
  defer>
</script>
```

### Liste blanche de domaines

Pour protéger le quota de votre compte et vous assurer que votre chat ne se charge que sur votre propre site, vous pouvez restreindre les domaines autorisés à exécuter le script.


 ![Screenshot Studio 1781949144938](/images/white-list-domains.webp)


- **Fonctionnement :** Dans les **Paramètres d'intégration** de votre chat, vous pouvez spécifier une liste de domaines autorisés (par ex., `mywebsite.com`). Le script du widget ne s'initialisera que sur ces origines.
- **Aucun domaine spécifié (par défaut) :** Si la liste des domaines autorisés est vide, le script du widget fonctionnera sur **n'importe quel domaine** (utile pour les tests locaux).

<Info>
  La vérification de la liste blanche de domaines est exécutée dynamiquement lorsque le script récupère la configuration du widget.
</Info>

## Afficher sur une page

L'option **Afficher sur une page** vous permet d'intégrer le chat complet directement dans une page web à l'aide d'un iframe. En ajoutant ce code à votre site web, blog ou page Notion, le chat apparaîtra directement dans le contenu de la page.

Cette option est idéale pour :

- Centres d'aide et pages FAQ
- Bases de connaissances et sites de documentation
- Portails de support
- Tableaux de bord internes

### Installation

Copiez et collez le code HTML suivant dans l'éditeur de votre site à l'endroit où vous souhaitez que le chat apparaisse :

```html
<iframe 
  src="https://attlas.so/chats/your-chat-slug?embed=true" 
  width="100%" 
  height="700px">
</iframe>
```

<Tip>
  Vous pouvez ajuster la largeur et la hauteur de l'iframe pour l'adapter à la mise en page de votre site.
</Tip>

<Warning>
  **Note de sécurité :** La configuration de la liste blanche de domaines (`allowed_domains`) **s'applique uniquement au script du widget**. Elle ne protège ni ne restreint le chargement direct de l'URL de l'iframe. Toute personne intégrant l'URL directe de l'iframe sur une page personnalisée pourra toujours afficher et interagir avec le chat.
</Warning>

## Canaux de distribution

L'onglet **Distribution** dans le panneau de partage vous permet de créer des canaux nommés pour suivre la provenance de vos conversations, par exemple, distinguer les utilisateurs arrivés via votre site web de ceux venus via une newsletter ou un QR code lors d'un événement.

### Qu'est-ce qu'un canal ?

Un canal est un libellé que vous définissez (par ex. `website`, `newsletter`, `event-june`). Lorsque vous le sélectionnez dans l'onglet Partager, Attlas ajoute automatiquement un paramètre `?origin=<channel>` à chaque méthode de partage que vous copiez :

| Méthode de partage | Avec canal sélectionné |
| :-- | :-- |
| Lien direct | `https://attlas.so/your-chat?origin=website` |
| Intégration iframe | `src="https://attlas.so/your-chat?origin=website"` |
| Script de bulle de chat | `<script ... data-origin="website" ...>` |

Cela vous permet de voir dans vos analyses quel canal a généré chaque conversation.

### Créer un canal


 ![Distribution channels tab](/images/distribution-channel_1.webp)


1. Ouvrez le panneau **Partager** et passez à l'onglet **Distribution**.
2. Cliquez dans le champ de saisie du canal et saisissez un nom (par ex. `newsletter`), puis appuyez sur **Entrée** pour l'ajouter.
3. Répétez l'opération pour ajouter autant de canaux que nécessaire.
4. Les canaux sont enregistrés automatiquement, aucune confirmation supplémentaire n'est nécessaire.

<Info>
  Les noms de canaux sont automatiquement convertis en slugs : les espaces et caractères spéciaux sont transformés en tirets et mis en minuscules (par ex. `My Blog Post` → `my-blog-post`).
</Info>

### Utiliser un canal lors du partage


 ![Distribution channels link sharing](/images/distribution-channel_2.webp)


1. Revenez à l'onglet **Partager**.
2. Si vous avez défini des canaux, une liste déroulante apparaît en haut de l'onglet.
3. Sélectionnez le canal qui correspond à la destination vers laquelle vous partagez.
4. Copiez le lien, le code de l'iframe ou le script, le paramètre `origin` est déjà inclus.

<Tip>
  Laissez le sélecteur de canal sur **Tous les canaux** lorsque vous souhaitez un lien générique sans attribution.
</Tip>

## Quelle option devriez-vous choisir ?

Chaque méthode de partage répond à un objectif différent :

| Option | Idéal pour | Suivi par canal | Liste blanche de domaines |
| :-- | :-- | :-- | :-- |
| **Lien direct** | Partage rapide sur n'importe quel canal numérique | Oui (`?origin=`) | Non |
| **Scanner et discuter** | Supports physiques et QR Codes | Oui (encodé dans le QR) | Non |
| **Bulle de chat** | Ajouter un assistant interactif à votre site web | Oui (`data-origin=`) | **Oui** (restreint le chargement du widget) |
| **Afficher sur une page** | Intégrer l'expérience de chat complète en ligne | Oui (`?origin=`) | Non (contourne les vérifications de domaine) |

Vous pouvez utiliser une ou plusieurs de ces options en même temps, créant ainsi plus d'opportunités pour les utilisateurs d'interagir avec votre chat.
