Une fois votre chat configuré et entraîné, il est temps de le partager avec le monde.

Attlas propose plusieurs méthodes de distribution pour que vous puissiez choisir celle qui correspond le mieux à votre audience, canal ou projet.

Pour accéder à ces options, cliquez sur le bouton **<kbd>Partager</kbd>** situé dans le coin supérieur droit de la plateforme.

![Options de partage 1](/images/sharing-options.webp)

---

## Options de partage

Voici les 4 principales façons de partager votre chat :

1. **Lien direct** : Idéal pour les réseaux sociaux, les e-mails et les applications de messagerie.
2. **Scanner et discuter (Code QR)** : Idéal pour les supports imprimés et les lieux physiques.
3. **Bulle de chat** : Un bouton flottant en bas de votre site web.
4. **Affichage sur une page (Iframe)** : Un widget intégré directement dans vos pages.

---

## Lien direct

Le lien direct vous permet de partager une version pleine page de votre chat. Vous pouvez l'utiliser sur différentes plateformes :

- **Réseaux sociaux** : Partagez des liens directement dans les publications ou les liens de bio (bio Instagram, LinkedIn, Linktree).
- **Messagerie** : Envoyez directement via WhatsApp, SMS ou Telegram.
- **E-mail** : Ajoutez aux newsletters, campagnes ou signatures d'e-mail.
- **Supports promotionnels** : Ajoutez aux présentations, PDF ou documents.

Copiez simplement le lien public généré depuis votre tableau de bord et partagez-le avec votre audience.

---

## Scanner et discuter (Code QR)

Connectez le monde physique à votre chat IA en générant un code QR personnalisé. Il est très efficace pour :

- **Supports imprimés** : Cartes de visite, flyers, catalogues et livrets.
- **Lieux physiques** : Menus de restaurant, tables, emballages ou affiches.
- **Événements** : Banderoles, stands de salon et présentations.

Une fois scanné, les utilisateurs seront redirigés directement vers votre chat dans leur navigateur mobile.

<Info>
  Si vous modifiez votre nom d'utilisateur dans la section **Profil**, vous devrez générer et télécharger un nouveau code QR.
</Info>

## Ajouter une bulle de chat

Ajoutez une bulle de chat flottante dans le coin inférieur de votre site pour que les visiteurs puissent interagir avec votre assistant sans quitter votre site.

 ![Attlas Bubble Chat Site](/images/attlas-bubble-chat-site.jpg)

<Note>
  L'option de partage en bulle de chat est disponible exclusivement sur les **forfaits Attlas Starter et PRO**.
</Note>

### Installation

Copiez le snippet de script et collez-le juste avant la balise `</body>` de fermeture de votre HTML :

```html
<script 
  id="attlas-widget" 
  data-chat-id="YOUR_CHAT_ID" 
  src="https://attlas.so/embed.js" 
  defer>
</script>
```

### Liste blanche de domaines

Pour protéger le quota de votre compte et vous assurer que votre chat ne se charge que sur votre propre site, vous pouvez restreindre les domaines autorisés à exécuter le script.

 ![Screenshot Studio 1781949144938](/images/white-list-domains.webp)

- **Fonctionnement :** Dans les **Paramètres d'intégration** de votre chat, vous pouvez spécifier une liste de domaines autorisés (par exemple, `mywebsite.com`). Le script du widget ne s'initialisera que sur ces origines.
- **Aucun domaine spécifié (par défaut) :** Si la liste des domaines autorisés est vide, le script du widget fonctionnera sur **n'importe quel domaine** (utile pour les tests locaux).

<Info>
  La vérification de la liste blanche de domaines est effectuée dynamiquement lorsque le script récupère la configuration du widget.
</Info>

## Affichage sur une page

L'option **Affichage sur une page** vous permet d'intégrer l'intégralité du chat directement dans une page web via un iframe. En ajoutant ce code à votre site web, blog ou page Notion, le chat apparaîtra directement dans le contenu de la page.

Cette option est idéale pour :

- Centres d'aide et pages FAQ
- Bases de connaissances et sites de documentation
- Portails de support
- Tableaux de bord internes

### Installation

Copiez et collez le code HTML suivant dans l'éditeur de votre site web à l'endroit où vous souhaitez que le chat apparaisse :

```html
<iframe 
  src="https://attlas.so/chats/your-chat-slug?embed=true" 
  width="100%" 
  height="700px">
</iframe>
```

<Tip>
  Vous pouvez ajuster la largeur et la hauteur de l'iframe en fonction de la mise en page de votre site.
</Tip>

<Warning>
  **Note de sécurité :** La configuration de la liste blanche de domaines (`allowed_domains`) **s'applique uniquement au script du widget**. Elle ne protège ni ne restreint le chargement direct de l'URL de l'iframe. Toute personne intégrant l'URL directe de l'iframe dans une page personnalisée pourra toujours afficher et interagir avec le chat.
</Warning>

## Quelle option choisir ?

Chaque méthode de partage répond à un besoin différent :

| Option | Idéal pour | Liste blanche de domaines |
| :-- | :-- | :-- |
| **Lien direct** | Partage rapide sur tous les canaux numériques | Non |
| **Scanner et discuter** | Supports physiques et codes QR | Non |
| **Bulle de chat** | Ajouter un assistant interactif à votre site web | **Oui** (restreint le chargement du widget) |
| **Affichage sur une page** | Intégrer l'expérience de chat complète en ligne | Non (contourne les vérifications de domaine) |

Vous pouvez utiliser une ou plusieurs de ces options en même temps, créant ainsi davantage d'opportunités pour les utilisateurs d'interagir avec votre chat.
