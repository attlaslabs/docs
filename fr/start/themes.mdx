Les thèmes ont été conçus pour vous aider à créer une expérience qui vous ressemble vraiment, plus belle, plus professionnelle et plus engageante pour tous ceux qui interagissent avec votre chat.

La section Thèmes vous permet de changer rapidement l'apparence de votre chat à l'aide de thèmes prédéfinis.

<Check>
  Allez dans `Settings → Themes` pour personnaliser l'apparence de votre chat.
</Check>

![Themes](/images/themes.jpg)

## Choisir un thème

Attlas offre actuellement **42 thèmes prêts à l'emploi**.

![Themes Attlas](/images/settings-themes.webp)

Chaque thème est livré avec sa propre combinaison de :

- Couleurs
- Styles visuels
- Design des boutons

Sélectionnez simplement un thème et toutes les modifications visuelles seront appliquées automatiquement à votre chat.

N'hésitez pas à explorer différentes options jusqu'à trouver celle qui représente le mieux votre marque, votre projet ou votre identité visuelle.

## Mode clair et mode sombre

En plus des thèmes, vous pouvez choisir comment votre chat est affiché.

### Mode clair

Utilise une interface propre et lumineuse.

### Mode sombre

Utilise une interface sombre et adapte automatiquement les couleurs du thème pour une meilleure expérience visuelle.

Vous pouvez basculer entre les modes à tout moment.

<Info>
  La section **Personnaliser**, qui vous permet de personnaliser les couleurs, la typographie, la mise en page et les images de fond, est disponible exclusivement sur les plans **Attlas Starter** et **Attlas PRO**.
</Info>
