A aba **Gatilhos** é onde você adiciona as perguntas e ações que aparecerão no centro do seu chat.

Os gatilhos funcionam como iniciadores de conversa. Eles despertam a curiosidade dos visitantes e os incentivam a interagir com seu chat nos primeiros segundos.

<Check>
  Para adicionar gatilhos, basta clicar na aba **"Gatilhos"** no menu lateral esquerdo.
</Check>

![Gatilhos do Attlas](/images/triggers_1.webp)

## Tipos de gatilhos

Você pode criar gatilhos personalizados usando diferentes tipos de conteúdo:

### Perguntas

Adicione perguntas que incentivem a interação e ajudem os usuários a descobrir mais sobre você, sua organização ou seu conteúdo.

**Exemplos:**

- Quem somos?
- Quais produtos oferecemos?
- Como funciona este serviço?
- Onde posso encontrar informações específicas?

As perguntas são uma das formas mais eficazes de iniciar conversas e guiar os usuários para os tópicos mais importantes.

![Gatilho de Pergunta](/images/trigger-editor-question.webp)

<Note>
  **Título** é o que os usuários veem antes de clicar. Torne-o curto, claro e envolvente. A **Pergunta** permanece oculta e só será enviada ao chat após o clique no gatilho.
</Note>

#### Gerar perguntas

Esta ação permite gerar perguntas a partir de um documento que já foi adicionado ao conhecimento do chat.

![Gerar Gatilhos](/images/generate-questions_1.webp)

Basta clicar no botão **Ação de IA** <kbd>Gerar Perguntas</kbd>** e o Attlas abrirá uma janela onde você pode escolher:

1. Selecione o documento do qual deseja gerar perguntas
2. Defina o número de perguntas que deseja gerar

Em poucos segundos, o Attlas gerará automaticamente sugestões com base no conteúdo do documento.

![Gerar Gatilhos](/images/generate-questions_2.webp)

Você pode então:

- Selecionar as perguntas que fazem mais sentido
- Remover aquelas que não deseja usar
- Escolher um gatilho pai no seletor **Pai de Destino** para agrupá-los sob um gatilho dobrável existente ou seção (ou deixar como **Raiz** para mantê-los no nível superior)
- Gerar novas sugestões se os resultados iniciais não atenderem às suas expectativas

Quando terminar, clique em `Adicionar` e pronto. As perguntas serão adicionadas imediatamente e podem ser testadas no **Playground**.

### Links externos

Adicione links para páginas importantes que você gostaria de promover.

Você pode compartilhar:

- Sites da empresa
- Páginas de destino
- Blogs
- Lojas online
- Páginas de cadastro
- Portfólios

Esta é uma ótima maneira de direcionar os usuários para conteúdo valioso e ações importantes fora do chat.

![Gatilho de Link](/images/trigger-editor-link.webp)

---

### Cartões

Crie cartões de link visuais para capturar a atenção dos usuários com uma imagem de capa grande.

Os cartões são perfeitos para destacar:

- Posts de blog e artigos
- Ofertas especiais ou promoções
- Notícias ou comunicados importantes
- Produtos ou serviços

O cartão inclui uma imagem de capa 16:9 e um título. Quando você adiciona uma URL, o cartão se torna clicável, redirecionando os usuários para sua página. Os visitantes também podem copiar facilmente o link diretamente do cartão.

<Info>
  O Attlas adiciona automaticamente parâmetros de rastreamento UTM para ajudar você a analisar o tráfego.
</Info>

![Gatilho de Cartão](/images/trigger-editor-card.webp)

Disponibilize arquivos para download diretamente através dos gatilhos.

Ideal para compartilhar:

### Mapas de localização

Compartilhe uma localização diretamente no chat para ajudar os usuários a encontrá-lo com mais facilidade.

Perfeito para empresas, escritórios, lojas, locais de eventos, restaurantes ou qualquer lugar que os usuários queiram visitar.

Basta adicionar um link de mapa e os usuários poderão visualizar a localização e obter direções com um único clique.

![Gatilho de Mapa](/images/trigger-editor-map.webp)

Os gatilhos são essenciais para o sucesso do seu chat, pois ajudam a quebrar o gelo e facilitam o início de uma conversa para os visitantes. Pense neles como o primeiro "olá" que seu chat dá aos novos usuários.

E se um usuário não quiser clicar em nenhum gatilho, tudo bem. Ele pode simplesmente digitar sua própria pergunta ou mensagem diretamente no chat.

  Use o botão **<kbd>Seção</kbd>** para criar seções e organizar seus gatilhos, perguntas e links em categorias

## Gatilhos Aninhados (Pai-Filho)

Para manter a interface do chat limpa e organizada, você pode aninhar gatilhos sob outros gatilhos para criar uma relação pai-filho.

### Gatilhos dobráveis

Apenas alguns tipos de gatilhos podem atuar como pai (dobrável) e conter gatilhos filhos:

| Tipo de Gatilho | Dobrável |
| --- | --- |
| Pergunta | Não |
| Link | Sim |
| Cartão | Sim |
| Arquivo | Sim |
| Google Drive | Sim |
| Vídeo do YouTube | Não |
| Canal do YouTube | Não |
| Spotify | Não |
| Mapa | Não |
| Email | Não |
| Telefone | Não |
| Seção | Sim |

### Como funciona

Quando um gatilho contém gatilhos filhos:

- **No Chat:** Clicar no gatilho pai expande-o inline para revelar seus filhos.
  - Se o pai for um gatilho de conteúdo (como Link, Arquivo ou Cartão), a expansão exibe uma linha de ação principal no topo (para visitar o link ou baixar o arquivo), seguida pelas perguntas filhas.
  - Se o pai for uma **Seção**, ele expande para exibir os itens filhos.

### Como configurar gatilhos aninhados

<Frame caption="Arrastar o gatilho para dentro do grupo">
  <img src="/images/trigger-drag-in.webp" alt="Arrastar o gatilho para dentro do grupo" />
</Frame>

<Frame caption="Arrastar o gatilho para fora do grupo">
  <img src="/images/trigger-drag-out.webp" alt="Arrastar o gatilho para fora do grupo" />
</Frame>

Você pode criar relações pai-filho de duas maneiras:

1. **Arrastar e Soltar:** Na lista de gatilhos, arraste um gatilho para a direita (recuando-o) para aninhá-lo sob o gatilho acima. Arraste-o de volta para a esquerda (desfazendo o recuo) para restaurá-lo como um gatilho de nível superior.
2. **Do Editor:** Abra a gaveta de edição de qualquer gatilho elegível para ser pai, role para baixo até a seção **Perguntas Sugeridas** e clique em <kbd>Adicionar Pergunta</kbd> para adicionar perguntas diretamente como filhos.
