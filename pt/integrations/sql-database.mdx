A integração com banco de dados SQL (compatível com PostgreSQL) permite conectar um banco de dados à sua sala de chat. Uma vez conectado, o Attlas será capaz de ler o esquema das tabelas autorizadas e formular consultas de leitura para responder com precisão às perguntas dos seus usuários.

---

## **🛠️ Como conectar**

1. Acesse **Configurações do Chat** \> **Integrações** \> **Banco de Dados SQL**.
2. Clique em **Conectar** e preencha os detalhes do seu banco de dados:
   - **Nome e descrição:** Um nome amigável e uma descrição para ajudar a IA a entender quais dados estão contidos.
   - **Moeda (opcional):** A moeda principal usada para valores monetários nas suas tabelas, ajudando a IA a formatar e interpretar dados financeiros.
   - **Host e porta:** O endereço do servidor do banco de dados e a porta (o padrão é `5432` para PostgreSQL).
   - **Banco de dados:** O nome do seu banco de dados.
   - **Usuário e senha:** Suas credenciais do banco de dados (idealmente somente leitura).
   - **Modo SSL:** Selecione `require` ou `disable`.
3. Clique em **Testar** para verificar as credenciais e ver quantas tabelas foram encontradas.
4. Clique em **Salvar** para finalizar.

## Preencha os detalhes da conexão

Clique no botão Conectar para abrir o formulário e preencha os seguintes campos:

| Campo | Exemplo de descrição |
| --- | --- |
| Nome | Um nome amigável para identificar essa conexão |
| Descrição | Uma descrição do que o banco de dados faz (ajuda o Attlas a entender o contexto dos dados) |

| Senha | A senha associada ao usuário |
| Modo SSL | O nível de segurança da conexão SSL |

## **⚙️ Gerenciando sua integração**

Clique em **Configurações** no cartão do Banco de Dados SQL ativo para gerenciar o seguinte:

- **Selecionar tabelas para compartilhar:** Você não precisa compartilhar tudo. Ative/desative quais tabelas ou views específicas a IA tem permissão para acessar.
- **Atualizar esquema:** Se você adicionar ou editar colunas/tabelas no banco de dados, clique em **Atualizar** para sincronizar as mudanças com a IA.
- **Desativar temporariamente:** Ligue ou desligue a integração a qualquer momento usando o botão de alternância.
- **Desconectar:** Clique em **Desconectar** para remover completamente a conexão com o banco de dados.

## **💡 Melhores práticas para o esquema do banco de dados**

Para ajudar o Attlas a escrever consultas precisas e realizar cálculos exatos, considere estas dicas de design de esquema:

- **Use tipos de coluna precisos:** Se você deseja que o chat realize cálculos, como somas, médias ou proporções, em dados numéricos, certifique-se de que essas colunas usem um tipo numérico, como `numeric`, `decimal`, `integer` ou `float`. Evite armazenar números como texto, como `varchar` ou `text`, pois isso torna os cálculos mais difíceis e menos precisos.
- **Adicione comentários nas colunas:** Use comentários do banco de dados nas suas tabelas e colunas. Adicionar uma breve descrição diretamente a uma coluna, por exemplo `COMMENT ON COLUMN users.status IS 'Active statuses: active, suspended, pending'`, fornece um contexto crucial que orienta o Attlas a escrever as consultas SQL corretas.

---

## **🔒 Melhores práticas de segurança**

Para manter seus dados seguros, recomendamos fortemente:

- **Use um usuário somente leitura:** Crie um usuário de banco de dados com apenas permissões `SELECT`. O Attlas não precisa de acesso de gravação (`INSERT`, `UPDATE`, `DELETE`).
- **Mantenha o SSL ativado:** Recomendamos definir `sslmode` como `require` para criptografar os dados em trânsito.
